# 士林電機 SPM-3 多功能集合式電表：Modbus 通訊與多機並接設定指南

根據士林電機官方 SPM-3 電表使用手冊，本設備支援 Modbus-RTU 協定，可透過手動修改 **「通訊位址 (Address)」** 實現多機並接監控。

---

## 一、 參數設定步驟 (按鍵操作)

### 1. 進入設定模式
* **進入方式**：在即時數據顯示介面，長按 **[SET]** 鍵（或同時按住 **[SET]** 與 **[▼]** 鍵）約 3 秒。
* **密碼驗證**：若畫面顯示 `PASS`，請輸入密碼（預設值通常為 **0000**），按 **[SET]** 進入。

### 2. 設定通訊站號 (Slave ID)
* **尋找參數**：利用 **[▲]** 或 **[▼]** 鍵切換，找到畫面顯示為 **`Addr`** 的項目。
* **修改數值**：按下 **[SET]** 鍵使數值閃爍，使用方向鍵調整。
    * **設定範圍**：1 ~ 247。
    * **並接規則**：第一台設為 `1`，第二台設為 `2`，以此類推。
* **確認儲存**：再次按下 **[SET]** 鍵鎖定數值。

### 3. 設定通訊格式 (所有設備必須一致)
除了 ID 之外，請確保以下參數與您的 PLC/網關設定完全相同：
* **鮑率 (Baud Rate)**：尋找 **`bAud`** 參數（建議設為 `9.6k` 或 `19.2k`）。
* **通訊格式**：尋找 **`Prto`** 參數（常用設定為 `n.8.1`）。

### 4. 退出並儲存
* 切換選單至顯示 **`End`**，按下 **[SET]** 鍵。電表會跳回測量畫面，此時新設定才會生效。

---

## 三、 多台並接配線注意事項 (RS-485)

1.  **接線方式**：請使用「手牽手」的方式串接（Daisy Chain），避免從中間分叉。
2.  **線材建議**：使用**帶遮蔽層的雙絞線**，並將 A(+) 與 B(-) 對接。
3.  **終端電阻**：若串接台數較多或距離較長，請在最後一台電表的 A、B 端之間並聯一顆 **120Ω 終端電阻**。

---

> [!常見問題]
> 	通訊不到？  檢查每一台電表的 `Addr` 是否重複。
> 	數據亂碼？  檢查 `Baud` (鮑率) 是否與主機一致。
