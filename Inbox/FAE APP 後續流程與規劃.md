- # FAE App 
	- ### 已確認項目
		- 設備必須聯網一次並確認
		- 大多情況會由FAE手機本身熱點進行聯網
		- 需要抓設備原先擬定好之電壓與Hz
	- ### 原本流程
		1. FAE掃描條碼
		2. 查詢設備資料並填寫設備與保固相關資訊
		3. 登陸保固
		4. 給客戶做安裝確認
	- ### 新增流程
		1. FAE掃描條碼【原流程】
		2. 協助設備聯網
		3. 查詢設備資料並填寫設備與保固相關資訊【原流程】
			2-1. 需多回傳設備原先擬定好之電壓與Hz
		4. 登陸保固【原流程】
		5. 幫設備做PCS初始化設定
		6. 給客戶做安裝確認

flowchart TD

H --> I{初始化成功?}
I -- 否 --> H
I -- 是 --> J[客戶安裝確認]

J --> K[流程完成]
